# Is it RIPE yet?

## Overview
This is my second project for General Assembly's Software Engineering Immersive program.  It is a full stack application to keep track of the ripeness of your bananas!

## Technologies used

* Node
* Express
* OAuth
* Javascript
* CSS
* HTML
